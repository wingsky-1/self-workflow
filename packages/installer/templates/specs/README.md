# Self-Workflow 规范（Spec）

> 规范是约束 Agent 行为的规则。Review Agent 在 Phase Gate 中逐条审查。
>
> **MUST** → 强制遵守，不通过则阻断 Phase Gate
> **SHOULD** → 建议遵守，不通过则产生警告
> **MAY** → 可选参考，仅记录

## 如何添加规范

在下方对应章节添加一条 `- ` 列表项即可。不需要了解 YAML 或编程知识。
